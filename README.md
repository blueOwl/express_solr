# express_solr

## Installation
Before installing, download and install Node.js.

`git clone https://github.com/blueOwl/express_solr`

`cd express_solr/`

`npm install`

## Configuration

set solr ip address, port and core name in `Server.js`

## Running
`nodejs Server.js `

or

`node Server.js `

default running on port 3000
